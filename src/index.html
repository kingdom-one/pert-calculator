Estimate your project's time with ease.

This app assumes that 1 day = 8 hours of work (like a normal workday), so if you set the dropdown to "days," the app will calculate that.

Since so many of us work part time, if you select "Weekly" a new field will appear so you can input your average hours-per-week worked.

How it works

A PERT calculator helps you give a project's time estimate by accounting for unknown variables. You give three time estimates for each task: the optimistic time, the most likely time, and the pessimistic time. The PERT calculator then calculates the expected time for each task and the expected project duration.
